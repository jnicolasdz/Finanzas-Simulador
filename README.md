# ¿Como instalarlo?

Descargue los contenidos de este repositorio y con la versión `3.14` de python instalada, ejecute:

## Windows
```
python -m venv venv
venv\Scripts\activate
```
## Linux
```
python3 -m venv venv
source venv/bin/activate
```

Luego, ejecute:

```
pip install -r requirements.txt
```
